Pipeline Management (Opportunity & Engagement Manager)

Pipeline management consists of the Opportunity and Engagement Manager functions allowing staff members to quickly view, access and update their current opportunities and engagements.

Run the Opportunity/Engagement Manager

  1. Click the "Opportunity Manager"/"Engagement Manager" link located in the Sidebar (under "Quick Links".)
  2. Set the Opportunity/Engagement Filters
  3. <Apply Filters>

Delegates

Delegates can run pipeline reports for the staff members they support.

  1. In the filter area, change the "Show Records" field to "All Opportunities" or "All Engagements".
  2. Type the staff members name in the "Staff Member" field.
  3. Select the staff member from the drop list.
  4. Set other filters as applicable.
  5. <Apply Filters>

Updating Your Pipeline

You can update your pipeline directly from the Opportunity or Engagement Manager page. Just remember to <Save Changes> before exiting the page.

Customizing the Layout

  1. Click the "Opportunity/Engagement Manager" link located in the Sidebar (under Quick Links.)
  2. <Change Columns>
    • Add a Column - Select the column(s) from the "Available" list then click.
    • Remove a Column - Select the column(s) from the "Selected" list then click
  3. Change Column Order - Select the column(s) from the "Selected" list then click "Up" or "Down".
  4. <Save>

Note: Multiple columns can be selected by holding down <Shift> or <Ctrl>.
