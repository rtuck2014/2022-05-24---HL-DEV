Add & Edit Hours (FVA & SC)

The Time Record Manager enables users to track time for projects (opportunities, engagements, special projects/non-chargeable activities) and is comprised of several sections:

  • Weekly Entry Matrix
  • Time Clock Recorder
  • Summary Logs
  • Detail Logs
  • Weekly Overview

Weekly Entry Matrix (Recommended Method)

Note: You can only add hours to projects which you are staffed. Otherwise, the project will not show up in the pick list (in Step 3)

  1. Click the "Time Record Manager" tab
  2. Select the appropriate "Time Record Period" (defaults to current week)
  3. Verify you are in the "Weekly Entry Matrix" tab (default)
  4. Select a project from the pick list. Projects can either be an Opportunity, Engagement or Special Project (FVA) / Non-Chargeable Activity (SC)
  5. For the appropriate day of the week, enter the hours worked on that project
  6. Enter the appropriate activity for those hours worked (under the hours box):
    • FVA - select most appropriate activity from the provided pick list
    • SC - only one activity option should be used "Strategic Consulting"
  7. Enter Comments (FVA Only)
    • Required for all FVA Special Projects
    • Enter comments in the comment box.
  8. Auto-save feature commits updates as soon as you tab or click outside of the field you are updating.
  9. Repeat steps to add hours to a new project or to log new activity hours to an existing project (if you are adding additional hours to an activity already containing hours on a project, just update the number of hours in the box)

Example: Let's say you work 4 hours on a project in one day and during that time you split your time between 2 activities. You should have 2 activity entries for that project on that day for 2 hours each.

Time Clock Recorder

The Time Clock Recorder is a running clock tool used to help track time spent on current activities

  1. Select Project and Activity (or Comment)
  2. Click Start to begin the clock. Leave clock running while working on an Activity
  3. Select Pause as needed, or Finish once Complete
  4. Once Finish is selected, the record will save and clock will reset
  5. If time entry no longer needed or incorrect, select Reset rather than Finish. This will erase the record

Summary Logs

Easy to review entries for time period. Able to add time from bottom of page by selecting Date, Project, Activity, and entering Hours in box. Once details are entered, click "Add"

Detail Logs

Similar to Summary Log, except allows for editing of current week entries. Activity and Hours entered can both be edited.

Weekly Overview

Quick view of all Projects for the time period. Read-only
