Common FAQs

When I use the Global Search, the record doesn't show and I know it's in Saleforce?

The initial search displays recently used record areas. Try clicking at the bottom the page and see if the record appears.

How can I quickly update the stage of an Opportunity or Engagement?

Via Pipeline Manager

  1. Select "Opportunity Manager" or "Engagement Manager" in the Quick Links section of the Sidebar (left-side)
  2. Add the appropriate filters then <Apply Filters> Delegates - be sure to type and select the appropriate staff member from the "Staff Member" field and change the "Show Records" field to "All Engagements".
  3. Update the stage directly on the page
  4. <Save Changes>

Via Opportunity Page

  1. Lookup the opportunity or engagement
  2. Double-click the "Stage" field located in the "Information" section
  3. Update accordingly
  4. <Save>

I want to search for all Strategic Consulting opportunities?

  1. Go to the "Opportunity" tab
  2. <Advanced Search>
  3. Select "SC" from the LOB dropdown
  4. Click <Search>
