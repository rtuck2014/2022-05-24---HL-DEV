Gift Log

The Gift Log tracks the total gift amount of a recipient (contact) based on the calendar year according to the SEC rules.

Request Gift Approval

  1. <Request Gift Approval> (main contact screen)
  2. Enter Billing Details
    • Gift Type (Req)
    • Submitted For (Req)
  3. Enter Gift Details
    • Gift Name (Req)
    • Gift Value (Req) - If you are entering gift for multiple recipients, the gift value should be the cost per recipient
    • Currency (Req)
    • Reason for Gift
    • HL Relationship
    • Desired Date
    • Vendor
  4. Enter Gift Recipient(s)
    1. Enter the company name or contact name in the appropriate search fields
    2. <Search> (if you are adding several recipients from the same company, it may be quicker to search by company name)
    3. Check the box next to the contact(s) to add as recipient(s)
    4. <Add Recipients>
  5. <Add Gift to Select Recipient(s)> (once all recipients are added)
  6. <Submit Request> (or <Revise Request> to change gift details)

The system does not automatically add the current contact as a recipient.
