Add Campaign and Add Response

Add to Campaign and Update Response

  1. Lookup the Contact in Salesforce
  2. Hover over Quick Link

  3. Select <Add to Campaign>

  4. In the Lookup window
    1. Search Within = All Campaigns
    2. Type campaign name in search field then <Go!>

  5. Click the correct campaign name link (be sure to choose the most recent campaign invite)


  6. Update campaign member information
    1. Response Method
    2. Status
    3. Speaker/Panelist
    4. Response Comments
  7. <Save>
    8.  

Update Response Only

  1. Lookup the Contact in Salesforce
  2. Hover over Quick Link

  3. Find the correct campaign and select "Edit"

  4. Update campaign member information
    1. Response Method
    2. Status
    3. Speaker/Panelist
    4. Response Comments
  5. <Save>

Alternative Method (via Campaign)

Adding a contact to a campaign via Campaigns is only available to Marketing and select staff.

This option allows you to add several contacts and response status to a campaign at once. However, you will need to edit each contacts response individually to add additional information like response method and requested by fields.

  1. Type the campaign name in the Global Search
  2. Click the appropriate campaign link (Campaign Name column)
  3. <Manage Members>
  4. To add a contact (and response) to a campaign
  5. Select "Add Members - Search" from the pick list
  6. Step 1 - click
  7. Step 2:
    1. Last Name equals "enter last name"
    2. First Name equals "enter first name"
  8. <Go!>
  9. Select the contact(s)to update
  10. <Add with Status>
  11. Select the appropriate response from the pick list
  12. To add additional response information:
    1. Select "Existing Members" tab
    2. Lookup the contact
    3. Select the contact name link and edit additional response information
