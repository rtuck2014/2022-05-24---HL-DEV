Accessing Reports

In Salesforce, customized reports have been developed and released to staff members based on their role at HL. Access the report via the "Reports" tab.

Print Report

For a quick and easy print, try printing the report from the browser. See "Print Screen" in Tips and Tricks.

  1. Open report
  2. <Printable View>
  3. Open the Excel file that should have appeared in the downloads footer.

Export Report

  1. Open report
  2. <Export Details>
  3. Keep default values for "Export File Encoding" & "Export File Format"
  4. <Export>

What's the difference!? - Although the output for Print and Export look similar, there are some differences. See FAQs for a comparison.

Share a Report Folder

You can share a report folder with another staff member.

  1. Click the "Reports" tab.
  2. Click the "Pin" icon next to the report folder.
  3. <Share>
  4. Select to share with "Users".
  5. Type in the person's name to share.
  6. <Share>

Pin Report Folder to Top

You can pin a frequently used report folder to the top of the list for easy access.

  1. Click the "Reports" tab
  2. Click the "Pin" icon next to the report folder you wish to pin
  3. Select "Pin to top"
