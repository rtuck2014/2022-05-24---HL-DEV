Submit Expenses for Approval

This section will show you how to submit an expense for approval. (Refer to the FVA Expense Guidelines for policy details)

  1. Click tab. If it is not visible, add it by customizing tabs
  2. Select the LOB and Event Type. These two fields are required
  3. Click Create New Expense Form
  4. Click the magnifying glass to select the Requestor Name

Expense approval requests may only be submitted for yourself or the person for whom you are a delegate. An error message will be triggered if the request is submitted for another user. Administrative users may submit expense requests for officers by selecting the officer's name as the Requestor and selecting the Administrative user's name as the Event Contact.

  1. Enter the Related Campaign and External Guests. One of these must be filled in prior to submitting form
  2. Click the magnifying glass to search for the Sub Ledger Number. Search by the first few digits or by name.
  3. Click All Fields for best search results
  4. Click Save to save your work
  5. Click Submit for Approval, resolve any error messages
  6. The expense Requestor will receive an email
  7. The expense Requestor will receive an email after a decision is made by approver

  1. To export the approved request to PDF, click Export PDF
