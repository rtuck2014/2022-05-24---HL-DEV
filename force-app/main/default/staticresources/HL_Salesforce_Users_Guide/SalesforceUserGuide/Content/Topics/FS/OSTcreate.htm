Create Offsite Template

This functionality is available to the Financial Sponsor Coverage Group only.

Create the Template

  1. Navigate to the covered company in Salesforce
  2. Hover over the "Coverage Team" Quick Link
  3. Select the Coverage Officer (click Officer Name link)
  4. <New Offsite Template> (Offsite Templates section)
  5. Select the Record Type then <Continue>
  6. Information Section- populate fields accordingly
  7. CAPIQ and Officer Information Section - LEAVE BLANK - will automatically populate once saved
  8. <Save>

Add/Edit Industries

  1. Scroll down to the Industries section
  2. <Mass Edit>
  3. Check "Focus" and "Meeting" boxes appropriately
  4. <Save and Return>

Add/Edit Investment Companies

  1. Scroll down to the Investment Companies section
  2. <Mass Edit>
  3. Current investments show by default (uncheck to see all )
  4. To include a company, the "Include" column
  5. To edit an investment company, click "Edit"
  6. Make updates then <Save>
  7. <Save and Return> from the main investments page
