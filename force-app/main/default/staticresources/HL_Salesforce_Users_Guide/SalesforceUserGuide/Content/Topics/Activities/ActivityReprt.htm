Run the Activity Report

The report provides a "print-friendly" version of the activity.

  1. Lookup the contact
  2. Hover over the Activities "Quick-Link"
  3. Click "View" for the activity you wish to run the report
  4. Click <View Report>
  5. If prompted, login using your HL Credentials
