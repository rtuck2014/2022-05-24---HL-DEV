Racetrack Report

  1. Lookup the engagement.
  2. <Racetrack Report> (top of the page).
  3. Enter report criteria on the filter page:
    1. Group By
    2. Sort
    3. Order
    4. Report Type
      1. With Checkmarks - Progress columns utilize checkmarks to signify Y/N
      2. With Dates - Progress columns show actual dates
    5. Columns - Select report columns to display in report (<Shift> or <Ctrl> to select multiple columns)
    6. File Type - select report output (PDF or Excel)
  4. <Run Report>
  5. <Back To Engagement>
