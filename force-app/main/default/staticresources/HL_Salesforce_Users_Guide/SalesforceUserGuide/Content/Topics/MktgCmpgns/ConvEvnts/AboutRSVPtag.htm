About RSVP Tagging and Contact Record Changes

What is it?

Each event has a web page with a form where attendees can register for the event and/or update their personal information. Some of the attendees register from the email invitation we sent to them. These responses will get auto-tagged in SF.

Other attendees register by coming directly to the website or via a forwarded email. We cannot auto-tag these -- we need someone to identify or add them to SF, then to the event campaign, and then tag them.

We also don't let strangers on the internet update our SF contact data, which is why you are asked to process record changes.

Your Responsibility

Administrative Assistants assist in the RSVP tagging & contact record changing process. Tasks include:

  1. Get the CSV report for the event registrations (will likely arrive as a daily report email)
  2. Process the RSVP tagging for all entries that need manual processing
  3. Process the contact record changes for all entries with such changes indicated in the CSV file

For detailed steps, please see Process RSVP Tagging and Contact Record Changes.
