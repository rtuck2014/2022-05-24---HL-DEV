Add New Opportunity

Watch the How-To Video

NOTE: The company(s) involved in the opportunity MUST be in Salesforce. It is suggested to search Salesforce before you start the opportunity creation process.

Step 1 - Enter Basic Opportunity Information

  1. Select the "Opportunities" tab
  2. <Add Opportunity>
  3. Select the "Record Type" then <Continue>
  4. Enter all required opportunity field information (I = Required Fields)
  5. <Save>

Step 2 - Add Additional Client/Subject Information

  1. Click <Add Client> or <Add Subject>
  2. Search for the Client/Subject . If the company does not exist, click the "New Company" tab and add the company.
  3. Check the select box next to the appropriate company(s).
  4. <Add Selected>
  5. Close the window "x".
  6. Indicate if any interests of Clients/Subjects are adverse to any other clients/subjects using the drop list.
  7. <Save & Close>

Step 3 - Build the HL Internal Team

  1. Type the staff name in the "Staff" field.
  2. Select the appropriate name from the automatic drop list.
  3. Check all roles that apply to that staff member.
  4. <Save>
  5. Repeat until all staff members are added.
  6. <Save>
  7. <Return to Opportunity>

RESTRICTED LIST UPDATE: If you select "YES" to MNPI when entering an opportunity, a notification is automatically sent to Compliance and no further action is needed. Do not manually add the company via the Restricted List button as a duplicate request will be created.
