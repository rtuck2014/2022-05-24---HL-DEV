Add Members to a Campaign

via Report

The following procedure allows you to add members via a report and is best if you if you wish to add a large number of members to a campaign from a previous campaign or from a report of a group of members (like all contacts that work for Boeing in Chicago.)

This process is only available to Marketing Department staff members. For non-marketing staff, please refer to the section Add a Contact to a Conference or Event.

  1. Click the "Reports" tab
  2. Click on Marketing Reports folder (left hand side)
  3. Open a pre-existing report or create a new custom report (see How to Print a Campaign List)
  4. <Customize>
  5. Search for campaign by clicking next to Select Campaign
  6. Search within "All Campaigns" then type in the campaign key words
  7. <Run Report>
  8. <Add to Campaign>
  9. In the Campaign field, click and search for the campaign you wish to add the members to
  10. Enter the Member Status (typically leave as default "Nominated for Invite" for conferences or events or "Queued" for mailing lists
  11. Existing Members
    • Do not override the member status - will not update the status of a member that has already been added to the campaign
    • Override the member status - if a member of this report is already part of the campaign, it will override their current member status with the member status you selected in step 10
  12. <Add to Campaign>

Direct via Campaign

This process is best used if you wish to update a small group of members directly from the Campaign area. The advantage to this process is that you can see all the contacts associated to the campaign.

  1. Go to the "Campaigns" tab
  2. Scroll down to the Campaign Members section where you can view all members of the campaign
  3. <Manage Members>
    • Add Members - Search - Search across all Salesforce contacts and add based on contact attributes like company name, title, etc. (e.g. Add all contacts that work for Wells Fargo with a title of Director).
    • Add Members - Import File - Do Not Use
    • Edit Members - Search - Search contacts within the current campaign and edit their current campaign member status. (e.g. Update a select group of members in a conference campaign from "Nominated for Invite" to "RSVP-Yes")
    • Update & Add Members - Import File -This allows you to add members via import from a .csv file. Please contact Marketing-DirectResponse@HL.com
  4. Depending on your selection in step 4, follow steps outlined in the below sections:

Add Members via Search

  1. Select "Add Members - Search" (continued from step 4 of Direct via Campaign)
  2. In Step 1 section, check "Contacts" (defaults to "Leads")
  3. Enter your search filter criteria (e.g. last name, first name, company, etc.) or skip to step 3 to pull entire member list
  4. <Go>
  5. Check the contacts to add
  6. Click <Add with Status> and select a Response Type

Add Members via Import

  1. Select "Update & Add Members - Import File" (continued from step 4 of Direct via Campaign)
  2. <Update & Add Campaign Members>
  3. In step 6 on the screen click <Choose File> to select the .csv file
  4. In step 8 on the screen, select the appropriate Response Type from the pick list
  5. In step 10 on the screen, check the box "Trigger workflow rules for new and updated records
  6. <Next>
  7. In Step 2 of 3 screen, map the Import Field with the appropriate Salesforce fields
  8. <Next>
  9. <Import Now!>

Edit Members via Search

  1. Select "Edit Members - Search" (continued from step 4 of Direct via Campaign)
  2. Enter your search filter criteria (e.g. last name, first name, company, etc.) or skip to step 3 to pull entire member list
  3. <Go>
  4. Check the contacts to update or remove then click <Remove> or select a Response Type from the <Update Status> pick list
