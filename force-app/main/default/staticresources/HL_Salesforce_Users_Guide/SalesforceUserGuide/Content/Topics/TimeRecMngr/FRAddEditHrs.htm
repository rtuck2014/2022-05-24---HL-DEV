Add and Edit Hours (FR)

The Time Record Manager enables users to track time for opportunities, engagements, and special projects.

View the video: Time Tracking in Salesforce (FR)

Add Hours

  1. Click the "Time Record Manager" tab (at top)
  2. Select the "Time Record Period" (defaults to current week)
  3. Select a project from the pick list
  4. Enter hours in boxes (round to nearest 1/2 hr, e.g. 2, 5.5, 10.5, 3)
    • Wk Day = Weekday Hours
    • Wk End = Weekend Hours
  5. Click <Add>

Edit Hours

  1. Click the "Time Record Manager" tab (at top)
  2. Select the "Time Record Period" (defaults to current week)
  3. Select a project from the pick list
  4. Update Weekday and Weekend hours accordingly (round to nearest 1/2 hr, e.g. 2, 5.5, 10.5, 3)
  5. You are done, no need to save (auto save function)

If you wish to remove all project hours for a given time record period, click in the project row.
