Searching/Looking Up Records

Watch the How-To Video

Global Search

The best way to lookup records is via the Global Search. Just type in a keyword in the box then click search.

Can't find the record you are looking for, try clicking at the bottom of the results page.

Keywords you can search by include (but not limited to):

  • Last Name or First Name
  • Company Name
  • Client/Subject
  • Project Name
  • Case #/Prospect ID
  • etc.

Global Search Options

You can drill down your results further by adding searching within the search.

Advanced Search

Another search options Within the Companies, Contacts, Opportunities and Engagement tabs, there is an "Advanced Search" option. This search feature allows you to drill down your search within each area using various field values.

  1. Click "Companies", "Contacts", "Opportunities" or "Engagements" tab.
  2. Enter your search criteria in the available fields. Expand the search field by clicking "Show Advanced Search" link.
  3. <Search>
