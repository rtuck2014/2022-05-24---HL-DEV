Create an Activity Via Outlook Calendar Event

You can create a calendar event in Outlook and Salesforce will match up the email addresses of the recipients and create a sync'd activity in Salesforce.

Before you can sync calendar events: 1) You must be setup to sync Salesforce and Outlook (contact SalesforceSupport@HL.com for assistance) and 2) If an invitee is not in Salesforce, the activity will link to it once the contact is created in Salesforce.

  1. Create an Outlook calendar event as normal
  2. Click "Categorize" and select "Salesforce" as the category
  3. Click <Send>
  4. Once syncing is completed (allow 30 min - 2 hrs):
    1. The activity will be created in Salesforce for the contact emails matched up from the recipient list
    2. An activity link will be created in the Outlook calendar invitation
