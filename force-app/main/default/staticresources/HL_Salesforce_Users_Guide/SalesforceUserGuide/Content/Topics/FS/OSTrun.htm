Run Offsite Template

This functionality is available to the Financial Sponsor Coverage Group only.

  1. Select "Financial Sponsors Reports" (under Report Links in the Sidebar)
  2. Select "Offsite Template Report"
  3. Login using your HL Network credentials
  4. Enter report options:
    1. Fiscal Year
    2. Officer
    3. Sponsor (hold Shift or Ctrl key to select multiple sponsors)
  5. <Run Report> (bottom-left)

You can also run the offsite template from the Coverage Team area:

  1. Lookup the covered company in Salesforce
  2. Hover over the "Coverage Team" Quick Link
  3. Click officer name link for which you want to run the offsite template
  4. Hover over the "Offsite Template" Quick Link
  5. Click the Offsite Template Name
  6. <Offsite Template Report>
  7. Follow the previous steps to running the report
