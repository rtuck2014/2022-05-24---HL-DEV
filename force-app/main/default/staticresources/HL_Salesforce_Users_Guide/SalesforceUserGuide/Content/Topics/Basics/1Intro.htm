What is Salesforce?

Salesforce is Houlihan Lokey's Customer Relationship Management (CRM) application built on the cloud based "Salesforce" platform. This system provides a central resource to manage contact, company, opportunity and engagement information.

Just getting started? Download the Salesforce Reference Sheet.

How to Access via Computer (In Office)

Click the Salesforce icon on your desktop. The system uses Single-Sign-On technology and will automatically log you in.

How to Access via Computer (Remote not on VPN)

  • Use this link to login: http://hl.my.salesforce.com
  • Your Username will need to be prefaced with “hlhz\” Example: hlhz\tturnbu
  • Enter your network password

How to Access via Mobile or Tablet

Download the Salesforce app from any popular app store.
