Filtering CSV File

To help keep track of the records you have already completed it is recommended to use the filtering function in Excel.

Access Filters in Excel: Sort & Filter > Filter > Click arrow in column header

First Report

  1. "update information" column - filter to show only "1s"

  2. "changed merge fields" column - filter to show all but " "

All Subsequent Reports

  1. "created at" - filter by date (filter picklist, use date filters > after...)

  2. "update information" column - filter to show only "1s" (see screenshot in earlier section)

  3. "changed merge fields" column - filter to show all but " " (see screenshot in earlier section)

Contact Record Changes

Registrants have updated their contact record information and it needs to be manually updated in Salesforce if:

  • "update information" column = "1" (see specific "updated" columns for changes, e.g. first name, last name, etc.)

    AND/OR

  • "changed merged fields" column contains data (changes listed directly in field)

Badge Changes - see EventBadges. If "will attend" = no, disregard Badge changes.
