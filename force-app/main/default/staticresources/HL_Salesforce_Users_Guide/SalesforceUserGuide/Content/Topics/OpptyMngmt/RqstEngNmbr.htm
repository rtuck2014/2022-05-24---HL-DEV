Request an Engagement Number

Important: The opportunity will no longer be visible in the system while waiting for an engagement number to be assigned! It will become available again (as an engagement) once an Engagement Number is assigned.

The Conflict Checks system is another application but access is available via Salesforce on the Opportunity page.

  1. Lookup the opportunity
  2. <Request Engagement Number> (top of the page)
  3. If you have not filled out all the required information, a pop-up window will list all remaining information needed

Note: Once an engagement, editing rights will be limited. If you find you are unable to edit a field that needs updated, please contact the appropriate LOB admin for assistance (see Support).
