Supervisor Role (FVA & SC)

Time Entry

Supervisors have the ability to view and edit hours for all staff within their respective group

  1. Click the "Time Record Manager" tab
  2. Select the "Time Record Period" (defaults to current week)
  3. Select staff name from list on left of page
  4. Make time entry updates directly in the boxes. Be sure to click out of the box after updating hours to trigger auto-save!

Rate Sheet Management (FVA - Dispute Resolution Consulting & SC ONLY)

Anytime a project is converted to an Engagement, a Rate Sheet will need to be added.

  1. Click the "Time Record Manager" tab
  2. Select “Rate Sheet Management”
  3. Select Engagement from “Engagement” pick list
  4. In the "Add Record" area, select a Rate Sheet from the "Select Rate Sheet" pick list:
    • FVA select "Schedule A”
    • SC select "HLSC - Schedule"
  5. Enter Start & End Date (select an End Date well past the Estimated Close Date to avoid having to update later)
  6. Select “Add”. Rates will now apply to any time entered for this project, and be used for the Invoicing
