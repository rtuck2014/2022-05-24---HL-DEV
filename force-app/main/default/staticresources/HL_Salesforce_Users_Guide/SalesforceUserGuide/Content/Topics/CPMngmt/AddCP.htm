Add Counterparties

  1. Lookup the opportunity/engagement
  2. <Counterparties> (top of the page)
  3. <Add Counterparties>
  4. Use of the three options below to search for counterparties (click to expand sections):
    1. Company Filter - <add AND row> or <add OR row> to add a search criteria rows then <Search>
    2. Existing Engagement - Click , search for an existing opportunity/engagement then <Search>
    3. Existing Company List - Click , search for the list then <Search>
  5. Select company(s) to add as counterparties
  6. <Add Select Records to...>
  7. Repeat steps to add additional counterparties
  8. <Back> (return to opportunity/engagement)

Need to create a company list? For instructions, please refer to Company Lists.
