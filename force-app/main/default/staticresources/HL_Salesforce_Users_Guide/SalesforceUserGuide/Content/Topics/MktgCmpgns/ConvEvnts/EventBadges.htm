Event Badges

  1. Lookup the contact in Salesforce
  2. Scroll to the "Event Badges" section
  3. Select to use current contact information or customize the badge information:
    • Use current information - Check the "Copy From Contact Detail" box (double-click).
    • Customize badge information - Double-click to edit the "Badge First Name", "Badge Last Name" and/or "Badge Company" fields.
  4. <Save>
